Vinculum Movie Search API Instructions

Vinculum User Guide

Singletons > To query for singleton endpoints in the URL

http://localhost:3000/api/"movie name"
- this will query movies by name (data from 1927 to 2017)
- Ex: localhost:3000/api/Coco

http://localhost:3000/api/"category"/"year-number"/true
- this will query movies by category that have won
- Ex: localhost:3000/api/Writing/1999/true

Collections > To query for our collections through the URL

http://localhost:3000/api/"category"/"year-number"
- this will return an array of movie names
- Ex: localhost:3000/api/Cinematography/2000
